  ### Pjoject 01. Simpltron (미니 기계어 인터프리터)
     고급 언어 프로그래밍이 아닌, 컴퓨터를 '완전히 열고' 내부 구조를 살펴봅니다.
     컴퓨터의 작동 원리를 직접 프로그래밍하여 작성하였습니다.
     LOAD, STORE, INPUT, OUTPUT, CONTROL 관련 명령어들을 직접 입력합니다.
